# JavaScript String Casing

## Methods
- `toUpperCase()` - converts to UPPERCASE
- `toLowerCase()` - converts to lowercase

## Usage
```javascript
var name = "John";
name.toUpperCase(); // Returns "JOHN"
name.toLowerCase(); // Returns "john"
```

## Important
- Original variable unchanged
- Must save result: `name = name.toUpperCase();`

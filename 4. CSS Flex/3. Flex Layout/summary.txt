# Flex Layout Summary

## Core Concept
Flexbox properties are divided into:
- **Parent (Container)**: Properties applied to the flex container
- **Child (Items)**: Properties applied to individual flex items

Understanding this distinction is essential for proper implementation.

## Key Properties

### Parent (Container) Properties
- **`flex-wrap`**: Controls wrapping (`nowrap`, `wrap`, `wrap-reverse`)
- **`justify-content`**: Aligns items along main axis (`flex-start`, `center`, `space-between`, etc.)
- **`align-items`**: Aligns items along cross axis (`flex-start`, `center`, `stretch`, etc.)
- **`align-content`**: Aligns wrapped lines (only works with `flex-wrap: wrap`)

### Child (Item) Properties
- **`order`**: Controls item sequence (default: 0, accepts any integer)
- **`align-self`**: Overrides container's `align-items` for individual items

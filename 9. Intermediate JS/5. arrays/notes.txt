FIZZBUZZ & ARRAYS

FizzBuzz Problem:
Print numbers 1-100, but:
- Multiples of 3: "Fizz"
- Multiples of 5: "Buzz" 
- Multiples of both: "FizzBuzz"

Why Important:
Classic interview question that tests basic coding skills.

Array Methods:
push() - adds items to end of array
pop() - removes last item from array

var output = [];
output.push(1);           // [1]
output.push("Fizz");      // [1, "Fizz"]

Basic Function Setup:
var output = [];
var count = 1;

function fizzBuzz() {
    output.push(count);
    count++;
    console.log(output);
}

Modulo Operator (%):
Checks divisibility - returns 0 if divisible
15 % 3 === 0    // true
15 % 5 === 0    // true

Complete Solution:
function fizzBuzz() {
    if (count % 3 === 0 && count % 5 === 0) {
        output.push("FizzBuzz");
    } else if (count % 3 === 0) {
        output.push("Fizz");
    } else if (count % 5 === 0) {
        output.push("Buzz");
    } else {
        output.push(count);
    }
    count++;
    console.log(output);
}

Key Points:
- Check FizzBuzz condition first (order matters!)
- Use else if to avoid multiple outputs
- Always increment count
- Modulo (%) checks divisibility

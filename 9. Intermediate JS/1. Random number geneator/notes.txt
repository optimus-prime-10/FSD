RANDOM NUMBER GENERATION IN JAVASCRIPT
=====================================

MATH.RANDOM() BASICS
• Generates floating-point numbers from 0 to <1
• Returns different values each execution
• Uses pseudo-random algorithms (not truly random)

RANGE CONVERSION
To get numbers in specific ranges:
1. Multiply by range size
2. Use Math.floor() to remove decimals  
3. Add offset to shift range

Example (1-6 dice roll):
var dice = Math.floor(Math.random() * 6) + 1;
